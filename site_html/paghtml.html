Introdução

Básicamente o HTML (Linguagem de Marcação de Hipertexto) que atualmente está na 5° verçãoé um tipo de HyperTexto usado para a criação de páginas web que pode ser intepretada pelos navegadores. Com ele é possivel inserir geralmente através de tags textos, imagens e vídeos atravês dos HyperTextos.

Sintaxe

A sintaxe básica do HTML é bem simples. em qualquer arquivo html inicialmente escrevemos o texto

Com essé codigo base, podemos desenvolve nosso site, independente do quão complexo seja, certamente ele parte do mesmo princípio. Apartir dessa sintaxe básica, é possivel sair de um simples hello world para um site completamente modificado e diferente de qualquer outro já feito. Dentro da tag <body></body>, você vai colocar oque você deseja que apareça na tela do navegador como parágrafos, imagens, links entre outros, enquanto no cabeçalho, são inseridas as configurações dá página como o título por exemplo.

Imagens

Para adicionarmos uma imagem à página, basta utilizar a tag <img>, dentro dela será inserido os atributos src="" que indica onde e qual é o arquivo, o
alt="" que serve como uma "identificação" entre outro como o width="" e o height="" que servem para definir largura e altura respectivamente.

Ordenadas

As listas ordenadas são usadas para enumerar os elementos e para isso usam as seguintes tags:

Tags Oque faz
<ol></ol> Cria a lista ordenada
<li></li> Itens da lista
